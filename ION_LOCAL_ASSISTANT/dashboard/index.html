ION Локальный Помощник

Режим: ТОЛЬКО ПРОВЕРКА / DRY RUN

Эта панель — безопасный центр управления. Она не удаляет файлы и не запускает live-действия. Для запуска действий используй CMD-меню или ярлыки.

Файлы

Проверка папок, дубликатов, больших файлов и пустых папок.

scripts\ION_MENU_RU.cmd

Фотографии

Поиск и группировка фото без перемещения и удаления.

photo_organizer_dry_run_v1.py

Проекты

Определение папок проектов: ECOM, код, документы, фото.

project_detector_v1.py

AI помощник

Русский локальный чат-помощник с оценкой риска.

scripts\start_ai_chat_ru.cmd

Отчёты

Все результаты сохраняются в папку:

storage\reports

Память

История действий, задач и AI-диалогов.

storage\memory

Как запускать

Открой Windows CMD в папке ION_LOCAL_ASSISTANT и запусти:

scripts\ION_MENU_RU.cmd

Правило безопасности

Любое действие, которое меняет файлы, требует отдельного подтверждения оператора.
