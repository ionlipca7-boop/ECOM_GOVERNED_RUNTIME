## Stable eBay Listing Foundation

Goal:

- strong first photo
- enough photos
- strong German SEO title
- complete item specifics
- clean HTML description
- valid category
- correct shipping/payment/return policies
- safe controlled publish/revise
- readonly verify

Already PASS or partial:
- controlled publish
- controlled revise
- token guard
- readonly verify
- photo intake
- quality baseline
- Telegram runtime

Missing but highest priority:
- SEO title engine
- item specifics engine
- clean HTML description engine
- first-photo ranking
- competitor comparison
- quality scoring

STOP CONDITION:
A listing should visually compete with strong marketplace sellers.
