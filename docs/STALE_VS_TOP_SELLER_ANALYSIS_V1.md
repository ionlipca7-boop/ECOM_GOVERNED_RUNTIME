# STALE_VS_TOP_SELLER_ANALYSIS_V1

## Purpose
Compare dead/stale listings against strong 500+ sold listings and extract rules for future agents.

## Key finding
Old is not automatically bad. A listing becomes stale when it has no engagement for a long period: no views, watchers, offers, messages, or sales.

## Dead listing signals
- 0 or very low views
- no watchers
- no offers/messages
- no sales
- weak title search match
- wrong or weak category
- missing item specifics
- poor first photo
- low photo count
- unclear description
- price not based on sold data
- duplicate/commodity product with no advantage

## Top seller signals
- 100/500/1000+ sold
- strong sales history
- multi-quantity listing
- restocked quantity
- preserved listing authority
- clean first photo
- many high-quality photos
- complete item specifics
- searchable title
- clear price/shipping/returns
- strong seller trust
- long-living listing with engagement

## Diagnosis order
1. Visibility problem: no views/impressions
2. Click problem: views but no watchers/clicks
3. Conversion problem: views/watchers but no sales
4. Offer problem: sales but weak margin or slow velocity

## Engine rules
- Do not blindly delete old listings.
- Preserve successful listing history when possible.
- For stale listings, first improve title/category/specifics/photos.
- Use Product Research/Terapeak/sold data before pricing changes.
- Compare against top sellers before revise.
- Build dry revise plan, then approval gate.
