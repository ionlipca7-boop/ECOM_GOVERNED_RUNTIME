PART 1 — EBAY CUSTOMER WORKFLOW LESSONS:
1. Defective cable buyer case:
- Buyer first had an item-not-received/delivery issue, then reported a broken cable with photos.
- Best response strategy: calm German message, ask for additional proof/photos/video, ask buyer to close/resolve the delivery case if item arrived, then handle defect separately.
- Do not admit fault before verification.
- For low-value items, offer partial refund or replacement to avoid negative feedback.
- Example partial refund strategy: offer 3 EUR for a damaged cable when total item value is low.

2. Pre-sale compatibility buyer question:
- Buyer asked if charger/cable works with Samsung Galaxy S25 5G and Samsung Galaxy A56 5G, whether it is original, and cable length.
- Safe response strategy: confirm only supported compatibility, state cable length, and avoid claiming Original/OEM unless truly original.
- Use wording: kompatibles USB-C Schnellladegerät mit Kabel.
- Always provide German message plus Russian explanation for Ion.

3. Soft discount / Preisvorschlag workflow:
- Buyer interested in 9.99 EUR listing.
- 5% discount calculation: 9.99 EUR - 0.50 EUR = 9.49 EUR.
- Safe German offer message: offer 5% Rabatt, price 9.49 EUR instead of 9.99 EUR.
- Future agent should calculate discount automatically and prepare eBay Preisvorschlag text.

4. Negative feedback / delivery dispute workflow:
- Buyer left negative feedback claiming no delivery/no tracking.
- Seller had tracking number A00613FI6500000007D6, sent to eBay Packstation address, tracking added to eBay, messages sent, buyer did not respond, parcel did not return.
