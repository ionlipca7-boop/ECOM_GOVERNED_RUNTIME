## Route 06 Canonical Purpose

Goal: find products with real demand, clear buyer problem, clean margin, low risk, listing opportunity, and product-line potential.

## Layer Contract

0 INPUT: creates product card with name, source, link, price, MOQ, shipping, category, photos, primary idea. No decision.

1 QUICK SAFETY FILTER: blocks obvious dangerous/high-risk goods such as counterfeits, risky electronics, weapons, chemicals, high-risk batteries, too heavy/big items, legal risk. If unsure -> NEEDS REVIEW.

2 DEMAND DISCOVERY: checks if people search for the problem or product. Outputs DEMAND_STRONG, DEMAND_MEDIUM, DEMAND_WEAK, DEMAND_UNKNOWN.

3 MARKET PROOF: checks sold listings, similar Amazon products, competitors, price, sell-through if available, market saturation. Outputs MARKET_PROOF_CONFIRMED, MARKET_PROOF_PARTIAL, MARKET_PROOF_WEAK, MARKET_PROOF_UNKNOWN.

4 BUYER BEHAVIOR: checks why buyer would buy using frustration, daily exposure, problem cost, emotional buy, decision friction, demo score, before/after score.

5 PRODUCT / PROFIT + RETURN RESERVE: calculates supplier price, China shipping, Germany shipping, packaging, eBay fees, ads 6-8%, returns, defects, misunderstandings. Profit is not PASS until return/defect reserve is included. Outputs PROFIT_PASS_AFTER_RESERVE, PROFIT_FAIL_AFTER_RESERVE, PASS_IF_BUNDLE.

6 VALUE SIGNAL / LISTING: checks whether listing can beat competitors through title, item specifics, buyer keywords, value words, photos, description, package truth, eBay algorithm fit. UD24 lesson: Title alone is not enough; Title + Item Specifics + Photos + Description + Package Truth + Delivery/Ads logic = Value Signal.

7 BUSINESS LADDER: checks whether the product can become a line: product ladder, expansion, multi-location, repeat purchase, bundle potential, ecosystem, replace/upgrade potential, season resistance.

8 OFFER STRUCTURE: decides selling structure: single listing, variation listing, bundle listing, replacement listings, premium listing, gaming/home office/travel versions, watch-only. No duplicate listing; each listing needs distinct buyer intent.

9 CRITIC AGENT: raises risk flags: too many competitors, price war, weak margin, returns, weak quality, bad glue, false specs, counterfeit brand, compatibility issues, market may die, easy copy. Critic does not decide final outcome.

10 DECISION BOARD: final decision only by board. Options: STRONG TEST, TEST SMALL LOT, WATCH 7 DAYS, WATCH 30 DAYS, NEEDS MORE DATA, REJECT, REPLACE / UPGRADE. Rule: NO SINGLE-AGENT FINAL DECISION except hard safety block.

11 MEMORY / LESSON: saves product, evidence, decision, risks, calculation, what later worked/failed, mistakes.

12 MASTER TEACHER / RULE ENGINE: compares forecast vs real result, finds agent mistakes, updates weights, reserves, SEO methods, rules, lessons.
