## Purpose

Routes 01 and 02 must support a German buyer-facing HTML shop block inside eBay descriptions.
Route 03 will later optimize and test this block.

The block helps buyer see:
- Ion shop;
- compatible products;
- accessories;
- replacements;
- upgrades;
- bundles;
- alternatives;
- selected extra products from the shop.

## Language

Buyer-facing content: German only.
Ion Telegram preview: Russian only.
Internal fields may be technical.

## Core rule

The block is allowed only if it is useful, truthful and not confusing.
It must support the main product, not distract from it.
