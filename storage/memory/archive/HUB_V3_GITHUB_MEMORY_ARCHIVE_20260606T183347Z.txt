## Route 01 integration

Route 01 simple listing must prepare:
- German product description;
- German shop block if safe related products exist;
- Russian preview for Ion;
- reason list for selected related products;
- missing link/image/stock warnings.

Examples:
- charger -> cable;
- cable -> charger;
- adapter -> compatible cable;
- simple product -> shop category if no direct related item.

## Route 02 integration

Route 02 variant listing must prepare:
- German group description;
- German variant guide;
- variant-aware shop block;
- Russian preview for Ion;
- related products that do not confuse variant choices.

Examples:
- set listing -> replacement cable;
- charger+cable set -> extra cable;
- variant product -> shop category for other lengths, powers or connectors.

## Route 03 integration later

Route 03 Listing Studio must optimize:
- block layout;
- product selection;
- title and description connection;
- test versions;
- related-product strategy.
