## Listing statuses

Route 04 uses four status levels:
- ⚫ Нет данных
- 🟢 Хороший
- 🟡 Средний
- 🔴 Требует внимания

## Notification rules

Good listing: silent.
Medium listing: appears in summary.
Bad listing: separate attention card.
No-data listing: setup/check card only.

## Attention card fields

- listing_id
- sku
- marketplace_id
- title_ru_short
- title_de_current
- status
- reason
- suggested_route
- suggested_action
- stock_note
- finance_note
- last_check_note

## Ion language

All Route 04 cards for Ion must be Russian.
Buyer-facing listing content remains German.
