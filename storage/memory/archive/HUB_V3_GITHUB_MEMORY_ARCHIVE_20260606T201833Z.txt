## Telegram card rule

All Route 07 cards for Ion must be Russian.
Cards must show only what Ion needs to decide.

## Card types

- approval needed
- blocked case
- missing data
- risk warning
- report summary
- waiting decision
- finished decision
- repair needed

## Card fields

- case_id
- source_route
- reason
- short summary
- risk level
- suggested decision
- buttons
- expires_or_review_note

## Button rule

Buttons must be short and route-safe.
No hidden live action behind a vague button.
