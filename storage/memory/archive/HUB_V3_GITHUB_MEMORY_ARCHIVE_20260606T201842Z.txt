## Approval gates

Route 07 handles approval gates for:
- publish path
- update path
- price change
- quantity change
- photo change
- delete or cleanup request
- purchase decision
- supplier decision
- blocked/retry decision

## Core rule

High-risk actions need explicit Ion approval marker.
No vague approval.
No hidden live action.
No delete without separate delete gate.
No secret values in cards.

## Report gates

Reports must be short and Russian:
- what happened
- what is blocked
- what is safe
- what Ion should decide
- what next route will do
