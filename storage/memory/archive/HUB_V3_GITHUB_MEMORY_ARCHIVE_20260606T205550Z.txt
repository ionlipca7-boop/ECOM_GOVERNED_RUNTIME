## Route 05 — Финансы / склад

Role: truth center for stock and money.
Purpose: know what was bought, what is available, what sold, what returned, what cost money and what created profit.

Two cores:
05A Stock core: SKU, variant, stock pool, bought, sold, returned, reserved, available, damaged, lost, physical place, marketplace quantity links.
05B Finance core: purchase, shipping, packaging, marketplace cost, ads, refund, return, reserve, revenue, result before/after reserve.

Agents:
Purchase Intake, Site Collector, Stock Ledger, Variant Stock, Return Agent, Cost View, Result View, Stock Alert, Reorder View, Report View, Tax Summary, Handoff.

Important rules:
One SKU has one shared stock truth.
If item is sold anywhere, shared stock pool is reduced.
Other routes request data from Route 05, not overwrite it.
Russian Telegram cards show stock/money actions for Ion.
