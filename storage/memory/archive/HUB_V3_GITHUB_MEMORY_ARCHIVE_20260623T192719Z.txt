## 7. Route01 — New single product
Role: Single Product Vault Builder.
Flow:
Intake URL/photos/receipt/quantity/price/notes -> Product Vault Bootstrap -> Product Master -> Source Evidence -> Receipt/Cost Parser -> Finance Packet -> Stock State -> Photo Manifest -> Category Agent -> eBay Taxonomy Aspects -> eBay Metadata Policy -> Radar Evidence -> Listing Packet -> Page Reader/Field Map -> Browser Fill Plan -> Validator -> Route07 Ion Review -> live gate only after approval.

Agents:
Route01 Orchestrator, Product Vault Bootstrap Agent, Source URL Reader, Receipt/Cost Parser, Photo Intake Agent, Photo Quality Agent, Product Passport Agent, Category Agent, eBay Taxonomy Agent, eBay Metadata Policy Agent, Finance Intake Agent, Stock Intake Agent, Radar Request Agent, Claim Safety Agent, Listing Packet Builder, Browser Page Reader, Field Binder, Validator, Route07 Decision Card Agent, Teacher Feedback Writer.

Rules:
- If URL has variants but only one variant was bought, Route01 may still be correct.
- If single/variant unclear, create Route07 decision card.
- No Product Vault + finance + stock + source evidence means no eBay fill/publish.
