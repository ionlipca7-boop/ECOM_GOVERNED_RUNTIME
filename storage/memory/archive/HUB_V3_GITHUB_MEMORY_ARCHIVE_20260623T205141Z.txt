## 2. Missing route restored: Direct Marketplace / Browser Publication Route
Ion noticed that the visual screens show mostly internal packet creation and separate publishing, but the earlier project logic included direct marketplace access.
This must NOT be forgotten.

V3 must support multiple publication modes:

### Mode A — Direct Marketplace Browser Route (priority for eBay first)
Use when Ion wants to publish/fill directly inside marketplace page.

Flow:
Product Vault -> Listing Packet -> Route07 approval to open fill route -> Browser opens eBay/Seller Hub -> Page Reader reads visible fields -> Field Map created -> Binder maps packet fields to visible eBay fields -> Fill Agent fills visible fields only -> Validator checks page -> Ion reviews in browser -> Submit remains closed until explicit Route07 live approval.

Important: Direct marketplace route is NOT auto-submit.
It is direct visible fill / preview / no hidden click.

### Mode B — Internal Project Draft Route
Use when eBay/Seller Hub is unstable, fields are missing, or Ion wants to prepare everything inside ECOM OS first.

Flow:
Product Vault -> Listing Studio -> eBay packet -> local preview -> Route07 review -> later direct browser/API fill.

### Mode C — API / Inventory Route
Use only for controlled updates or API-safe cases.

Flow:
Product Vault -> eBay API adapter -> dry payload -> validator -> Route07 approval -> live write action only after explicit gate.

Examples:
- price update
- quantity update
- photo update through Inventory API
- read-only monitor actions

### Mode D — Manual / Recovery Route
Use when browser/API fail.

Flow:
Product Vault -> generate operator checklist -> Ion manually copies/fills marketplace -> system records result -> Route08 lesson.

## 4. Browser/Page Reader rules
Direct marketplace route requires:
- visible page only
- no hidden submit
- no auto-click on publish/revise/delete
- no save/submit without explicit Route07 marker
- before fill: confirm listing packet, product_key, SKU, title, photos, category, item specifics, price, quantity, shipping, returns, ads gate
- after fill: screenshot/readback validator
- if mismatch: STOP_SAFE_FIELD_BINDER_MISMATCH
- if eBay changes UI: STOP_SAFE_PAGE_READER_FIELD_MAP_REQUIRED
